Conceitos Básicos do HTML

Por que aprender a linguagem HTML?

HTML ou HyperText Markup Language é uma linguagem utilizada pelo navegador para se comunicar com os servidores que hospedam os arquivos que compõe a web (textos, imagens,vídeos,músicas,entre outros).

Componentes da Web

HTTP é o principal protocolo usado pelo navegador para se comunicar com os servidores.
Servidores são computadores que hospedam arquivos que compôe a web como citado anteriormente.
A internet é a maior rede de computadores do mundo.
Navegador é um programa executado nos computadores para exibir arquivos encontrados na web.

Tags principais

A tag <html> serve para indicar ao navegador que se trata de um arquivo HTML.
A tag <b> (bold) indica o uso de um traço mais grosso que o comum para dar realce às palavras.
A tag <em> indica o uso de uma linha mais fina e inclinada para dar ênfase.
A tag <a href="link">palavra a ser linkada<a/> é usada para criar um link em meio ao texto para acessar outra página.
A tag <img src="link" alt=texto de erro> serve para adicionar imagens, sendo uma tag vazia(sem fechamento).
A tag <br> serve para quebrar uma linha e também é chamada de tag vazia(está sendo usada agora para pular linhas.)
A tag <p> é usada para criar um parágrafo, e é chamada de tag em bloco, pois seu conteúdo fica dentro de uma caixa.

Como computadores são "burros"?

A força de expressão utilizada para entender o sentido da programação representa que o computador não consegue deduzir uma ambiguidade que muitas vezes nós conseguimos. Ou seja, podemos ser muito mais inteligentes que um aparelho, basta conseguir instruí-lo da melhor maneira possível. Pois um mínimo erro é capaz de estragar quando está na mão de um computador "burro". A explicação desse contexto segue segue no link a seguir: Os computadores são burros
